Standard

MACS Core v1.0

The MACS Core defines the organisational, decision-related and documentation minimum requirements for handling cryptographically secured digital assets. Core v1.0 is formally fixed and frozen.

Binding versions (PDF)

  • MACS Core v1.0 – English (Authoritative Translation) (PDF)
  • MACS Core v1.0 – German (Master Version) (PDF)

The binding status is shown on the Status page.

Structure of the MACS Core

MACS Core v1.0 contains the following sections:

  1. Purpose
  2. Definitions
  3. Scope
  4. Guiding principles
  5. Minimum requirements (Core Requirements)
  6. Subject matter of MACS
  7. Further development
