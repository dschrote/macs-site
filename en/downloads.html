Downloads

This page provides the binding versions of the Minimum Acceptable Crypto Standard (MACS) as well as related formal documents.

MACS Core

The MACS Core defines the minimum requirements of the standard. The version published here is the binding reference.

  • MACS Core v1.0 – German (Master Version) (PDF)
  • MACS Core v1.0 – English (Authoritative Translation) (PDF)

Annexes

Annexes provide thematic clarifications and additions to the MACS Core. No annexes have been published at this time.

Formal documents

The following documents record the formal fixing and status of the MACS Core.

  • Formal fixing statement – MACS Core v1.0 (German) (PDF)
  • Formal Freeze Statement – MACS Core v1.0 (English) (PDF)

Archive

Archived versions of earlier documents are not current and have no normative effect. The current binding status is shown on the Status page.
